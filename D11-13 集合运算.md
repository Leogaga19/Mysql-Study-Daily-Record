## 表的加减法
### 表的加法`UNION`
示例：
```
mysql> SELECT * FROM pro2;
+------+------------+--------------+--------+--------+------------+

| 0003 | 运动T恤    | 衣服         |   4000 |   2800 | NULL       |
| 0009 | 手套       | 衣服         |    800 |    500 | NULL       |
| 0010 | 水壶       | 厨房用具     |   2000 |   1700 | 2009-09-20 |
+------+------------+--------------+--------+--------+------------+
5 rows in set (0.00 sec)

mysql> SELECT * FROM pro;
+------+------------+--------------+--------+--------+------------+
| id   | name       | type         | sprice | pprice | rd         |
+------+------------+--------------+--------+--------+------------+
| 0001 | T恤衫      | 衣服         |   1000 |    500 | 2009-09-20 |
| 0002 | 打孔器     | 办公用品     |    500 |    320 | 2009-09-11 |
| 0003 | 运动T恤    | 衣服         |   4000 |   2800 | NULL       |
| 0004 | 菜刀       | 厨房用具     |   3000 |   2800 | 2009-09-20 |
| 0005 | 高压锅     | 厨房用具     |   6800 |   5000 | 2009-01-15 |

mysql> SELECT id, name, type FROM pro
    -> UNION
    -> SELECT id, name, type FROM pro2;
+------+------------+--------------+
| id   | name       | type         |
+------+------------+--------------+
| 0001 | T恤衫      | 衣服         |           #对重复的‘0001’‘0002’‘0002’进行合并，此处仅出现一次。
| 0002 | 打孔器     | 办公用品     |

| 0008 | 圆珠笔     | 办公用品     |

| 0010 | 水壶       | 厨房用具     |

```

#### 注意事项：
1.作为运算对象的计录的列数必须相同；
2.作为运算对象的计录的列的类型必须一致；
3.可以使用任何`SELECT`语句，单`ORDER BY`子句只能在最后使用一次。

### 包含重复行的机会运算————ALL选项
**`UNION`保留重复选项的办法就是使用`UNION ALL`语句**
示例：
```
mysql> SELECT name, type, id FROM pro
    -> UNION ALL
    -> SELECT name, type, id FROM pro2;
+------------+--------------+------+
| name       | type         | id   |
+------------+--------------+------+
| T恤衫      | 衣服         | 0001 |

| 叉子       | 厨房用具     | 0006 |
| 擦菜板     | 厨房用具     | 0007 |
| 圆珠笔     | 办公用品     | 0008 |

```
