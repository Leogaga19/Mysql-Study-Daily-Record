# 谓词
## MySQL谓词--BETWEEN范围查询

直接使用**BETWEEN将包含临界值**，`BETWEEN 50 AND 100 `即50≤X≤100。示例：
```
mysql> SELECT name, sprice FROM pro
    -> WHERE sprice between 100 AND 1000;
+-----------+--------+
| name      | sprice |
+-----------+--------+
| T恤衫     |   1000 |
| 打孔器    |    500 |
| 叉子      |    500 |
| 擦菜板    |    880 |
| 圆珠笔    |    100 |
+-----------+--------+
5 rows in set (0.19 sec)

mysql> SELECT name, sprice FROM pro
    -> WHERE sprice > 100
    -> AND sprice < 1000;
+-----------+--------+
| name      | sprice |
+-----------+--------+
| 打孔器    |    500 |
| 叉子      |    500 |
| 擦菜板    |    880 |
+-----------+--------+
3 rows in set (0.00 sec)
```

# 谓词--IN
**01--用作OR的简便用法`IN(值1，值2，……)`找出等于值1，值2……的数据**
```
mysql> SELECT name, pprice FROM pro
    -> WHERE pprice IN (320, 500, 5000);
+-----------+--------+
| name      | pprice |
+-----------+--------+
| T恤衫     |    500 |
| 打孔器    |    320 |
| 高压锅    |   5000 |
+-----------+--------+
3 rows in set (0.00 sec)

mysql> SELECT name, pprice FROM pro
    -> WHERE pprice NOT IN (320, 500, 5000);
+------------+--------+
| name       | pprice |
+------------+--------+
| 运动T恤    |   2800 |
| 菜刀       |   2800 |
| 擦菜板     |    790 |
+------------+--------+
3 rows in set (0.00 sec)
```

### 谓语EXIST
**EXIST英语意指：存在**
示例：选出“大阪店（000C）在售商品（product_id）的销售单价（sale_price）”。
```
mysql> SELECT id, name, sprice FROM pro AS P
    -> WHERE EXISTS (SELECT * FROM SP AS SSP
    -> WHERE SSP.id = "000C"
    -> AND SSP.pid = P.id);
+------+------------+--------+
| id   | name       | sprice |
+------+------------+--------+
| 0003 | 运动T恤    |   4000 |
| 0004 | 菜刀       |   3000 |
| 0006 | 叉子       |    500 |
| 0007 | 擦菜板     |    880 |
+------+------------+--------+
4 rows in set (0.00 sec)
```
